# 🤖 Discord RP & Moderations-Bot

Ein vollständiger Discord-Bot mit Moderation, Ticketsystem, RP-Einsätzen und mehr.

## 📋 Befehle

| Kategorie | Befehle |
|-----------|---------|
| 🛡️ Moderation | `/warn` `/warnings` `/kick` `/ban` `/timeout` |
| 🎫 Tickets | `/ticket` `/close` |
| 🚨 Einsatz (RP) | `/einsatz` |
| 🎖️ Dienstgrade | `/befördern` `/rang` |
| 📊 Statistiken | `/stats` `/serverinfo` |
| ⚙️ Utility | `/ping` `/hilfe` |
